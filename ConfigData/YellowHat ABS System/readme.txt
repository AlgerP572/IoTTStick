Load the YellowHat_ABS.json file from the node configuration page to configure the IoTT Stick
- using the LocoNet Loopback interface as command source
- connect to a YellowHat running port A as block detector inputs
- activate event handler to implement ABS based on the block detector input
- drive a LED chain with up to 80 LED's (1 on board, 64 for indicator panel, 12 for signals, 3 unused)
